# Nector-gamer
# NYC Exploration Game

A 3D web-based exploration game set in New York City built with Three.js.

## Features
- Explore a 3D city environment
- Basic character movement and jumping
- Vehicle system (placeholder)
- First-person perspective

## Setup
1. Clone this repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open http://localhost:3000

## Controls
- WASD: Move character
- Space: Jump
- Mouse: Look around

## Deployment
This project can be deployed on Render.com as a static site.
